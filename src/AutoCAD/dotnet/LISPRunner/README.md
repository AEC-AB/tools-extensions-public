# LISP Runner Help files

## How to use this Extension

1. Open a drawing in AutoCAD.
2. Select the LISP Runner extension from Assistant.
3. Choose a mode:
	- `SelectFile` shows the `Lisp Script` file picker for selecting a LISP file.
	- `Inline` shows a multi-line text box for entering Lisp code directly.
4. Run the extension.
