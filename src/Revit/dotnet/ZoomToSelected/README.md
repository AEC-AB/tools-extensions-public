# Zoom To Selected

## Description
This extension automatically zooms and centers the Revit view to fit all selected elements. 

## Features
- **Instant Focus**: Zoom to all selected elements in one click
- **Smart Padding**: Automatically adds appropriate spacing around selected elements
- **View Aware**: Works with the current active view, respecting view boundaries

## Usage
1. Select one or more elements in your Revit model
2. Run the extension through Assistant
3. The view will automatically zoom and center on your selection

## Tip: Add as a Revit Button
You can make this extension even more convenient by adding it as a button in Revit. Use the **shortcut feature in Assistant** to create a quick-access button in your Revit toolbar. This allows you to zoom to your selection with a single click, speeding up your workflow for frequently used operations.
